# 1.1 截圖契約

只使用永久規範指定、既有的最新 Simulator，一次一台：

- iPhone 6.9：`Screenshot-iPhone-6.9`
- iPad 13：`Screenshot-iPad-13`

每種裝置 6 張，繁體中文、無除錯疊層、無鍵盤遮擋、無購買完成假資料：

1. 首頁：新低高度中文 Banner、真正搜尋框、五大分類。
2. 圖卡快覽：Pinterest 式直式雙欄。
3. 搜尋結果：使用可重現關鍵字，顯示真正結果。
4. 100 個工具：工具目錄與搜尋。
5. 工具內容：查核、風險分數、現場筆記。
6. Pro：清楚說明一次買斷、解鎖內容、恢復購買。

- iPhone 6.9：`1320 × 2868`
